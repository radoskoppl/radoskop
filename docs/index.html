Czym jest Radoskop?

Bezpłatne narzędzie do monitorowania rad miejskich w Polsce. Przetwarzamy dane z BIP: frekwencję, głosowania, interpelacje, profile radnych i budżety. Kadencja 2024–2029.

Dane aktualizowane automatycznie po każdej sesji. Kod źródłowy na GitHubie (AGPL-3.0). Chcesz dodać swoje miasto? Otwórz issue albo pull request.
